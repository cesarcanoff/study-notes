# Aula 03 - Como usar textos e Links em HTML

### Texto

```html
<p></p>
```

### Links

```html
<!-- LINK PARA PÃGINAS -->
<a href="anyLink">Link</a>

<!-- LINK PARA E-MAIL -->
<a href="mailto:canoff.developer@gmail.com">E-mail</a>

<!-- LINK PARA ABRIR EM OUTRA ABA -->
<a href="anyLink" target="_blank">LINK</a>
```
