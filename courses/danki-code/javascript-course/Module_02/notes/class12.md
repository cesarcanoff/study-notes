# Aula 12 - Laços de Repetição - For, While e Do While

```jsx
// Laços de Repetição - For, While e Do While

/* WHILE */
let contador = 0;
while (contador <= 10) {
  console.log('WHILE: ' + contador);
  contador++;
}

/* FOR */
for (let i = 0; i < 10; i++) {
  console.log('FOR: ' + i);
}

/* DO WHILE */
let cont = 0;
do {
  console.log('DO WHILE: ' + cont);
  cont++;
} while (cont <= 10);
```
